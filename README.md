# AI-Powered-Embedded-Tracking-System
Real-time person tracking turret on the Arduino UNO Q. A 2-DOF pan/tilt servo with a 1080p camera and laser follows a person using on-board AI detection. A Kalman filter smooths the noisy 10-15 Hz detections and predicts target position from velocity, and a PD controller drives the servos without overshoot.
